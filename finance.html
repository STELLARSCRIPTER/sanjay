Investment Strategies

Description: This project explores various investment strategies, focusing on value investing, growth investing, and the debate between passive and active management. Real-world examples are provided to illustrate each strategy's effectiveness.

Advantages

  • Provides insights into different investment approaches.
  • Helps in understanding the risk and reward of various strategies.
  • Offers practical examples to guide investment decisions.

Tools Used

  • Excel (for financial modeling)
  • Python (for backtesting strategies)
  • Yahoo Finance API (for data extraction)
  • Tableau (for data visualization)

Approximately 8 weeks, including research, strategy development, and backtesting.
