Physics-Based Puzzle Game

Explore challenging puzzles with realistic physics interactions.

Project Overview

Description: This project involves creating a physics-based puzzle game where players solve puzzles using realistic physics interactions. The game includes various levels with increasing difficulty, where players must use physics principles to overcome obstacles and achieve goals.

Advantages

  • Engages players with interactive and challenging puzzles.
  • Provides a realistic physics experience, enhancing immersion.
  • Encourages problem-solving and critical thinking.

Tools Used

  • Unity Game Engine
  • C# Programming Language
  • Blender for 3D Models
  • Photoshop for Textures

Development Timeline

Approximately 5 months, including planning, development, testing, and polishing phases.
